# RealFutbol

Resultados de fútbol en tiempo real con foro global y autenticación con Google.

## Funcionalidades
- Resultados en vivo y por fecha
- Foro global en tiempo real
- Login con Google obligatorio para comentar
- Sistema de roles (desarrollador / moderadores)
- Moderación de mensajes

## Stack
- Frontend: HTML / CSS / JavaScript
- Backend: Firebase (Auth + Firestore)
- Hosting: Vercel
- API de fútbol (datos en tiempo real)
